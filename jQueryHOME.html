jQuery 자습서(Tutorial)

  • jQuery 는 JavaScript 라이브러리입니다.
  • jQuery 는 JavaScript 프로그래밍을 크게 단순화 합니다.
  • jQuery 는 배우기 쉽습니다.

"스스로 시험해보십시오" 각 장의 예

온라인 편집기를 사용하여 코드를 편집하고 버튼을 클릭하여 결과를 볼 수 있습니다.

ex

$(document).ready(function(){
   $("p").click(function(){
     $(this).hide();
  });
});
