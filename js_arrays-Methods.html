JavaScript Arrays Methods

JavsScript 배열은 단일 변수에 여러 값을 저장하는 데 사용됩니다.

배열을 문자열로 변환

JavaScript 메소드는 toString()은 배열을 쉼표로 구분 된 배열 값의 문자열로 변환합니다.

ex

var fruits = [ "Banana", "Orange", "Apple", "Mango"];
document.getElementById("demo").innerHTML = fruits.toString();

결과

Banana,Orange, Apple, Mango

조인() 메소드는 문자열에 모든 배열 요소를 결합한다.
그것은 toString()과 똑같이 동작하지만, 구분자를 지정할 수도 있습니다:

ex

var fruits = [ "Banana", "Orange", "Apple", "Mango"];
document.getElementById("demo").innerHTML = fruits.join(" * ");

결과

Banana * Orange * Apple * Mango
직접해보기

터지는 것과 밀기

배열을 사용하여 작업하면 요소를 쉽게 제거 하고 새 요소를 추가 할 수 있습니다.
이것은 터지는 것과 밀리는 것입니다.
배열에서 항목을 꺼내거나 항복을 배열로 푸시합니다.

Popping

pop()메소드는 배열의 마지막 한 요소를 제거한다.

ex

var fruits = ["Banana", "Orange", "Apple", "Mango"]
fruits.pop(); //Removes the last element ("Mango") from fruits
